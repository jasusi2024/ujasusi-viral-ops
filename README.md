# Ujasusi VIRAL_OPS
**Intelligence Content Amplification System**  
by [Ujasusi Blog](https://ujasusiblog.com)

Transforms Ujasusi Blog intelligence analysis articles into viral, platform-optimised social media posts across X/Twitter, LinkedIn, Substack Notes, and WhatsApp.

---

## Deploy to Vercel (5 minutes)

### Option A ‚Äî Drag & Drop (No account setup needed)
1. Go to [vercel.com](https://vercel.com) and sign up / log in (free)
2. Click **"Add New Project"**
3. Click **"Import Git Repository"** ‚Äî or drag this entire folder into the upload zone
4. Leave all settings as default
5. Click **Deploy**
6. Your app will be live at `ujasusi-viral-ops.vercel.app` (or similar)

### Option B ‚Äî Via GitHub
1. Push this folder to a GitHub repository
2. Connect the repo to Vercel
3. Deploy ‚Äî Vercel auto-detects Vite/React

---

## Local Development

```bash
npm install
npm run dev
```
App runs at `http://localhost:5173`

---

## Usage

1. Enter your **Anthropic API key** (get one free at [console.anthropic.com](https://console.anthropic.com/settings/keys))
2. Paste your article title and content
3. Click **Generate Viral Posts**
4. Copy each platform's output via the tabs

Your API key is stored only in your browser's session memory ‚Äî never logged anywhere.
